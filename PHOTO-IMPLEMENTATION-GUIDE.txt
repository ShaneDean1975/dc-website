STEP-BY-STEP PHOTO IMPLEMENTATION
----------------------------------

STEP 1: PREPARE YOUR PHOTOS
1. Take or source professional headshots
2. Crop to portrait orientation (4:5 ratio preferred)
3. Optimize file size (use tools like TinyPNG.com or Squoosh.app)
4. Create WebP versions for better performance

STEP 2: NAME YOUR FILES CORRECTLY
Save photos with SEO-friendly names:
✓ shane-dean-photo.jpg
✓ shane-dean-photo.webp
✓ chad-camper-photo.jpg
✓ chad-camper-photo.webp

✗ DON'T use: IMG_1234.jpg, photo.jpg, attorney.jpg

STEP 3: UPLOAD FILES TO YOUR SERVER
Upload to your website's root directory or /images/ folder:
- yoursite.com/shane-dean-photo.jpg
- yoursite.com/shane-dean-photo.webp
- yoursite.com/chad-camper-photo.jpg
- yoursite.com/chad-camper-photo.webp

STEP 4: VERIFY PHOTO PLACEMENT
The HTML files already have placeholders ready. Photos will automatically display once uploaded with the correct filenames to your server.

TROUBLESHOOTING
---------------

ISSUE: Photos not displaying
→ Verify filenames exactly match (case-sensitive on some servers)
→ Check file upload was successful
→ Ensure correct file path in HTML

ISSUE: Photos look pixelated
→ Use higher resolution source images (minimum 800x1000px)
→ Don't upscale small photos - use original high-res versions
→ Check image compression settings

ISSUE: Page loads slowly
→ Optimize image file sizes (target under 200KB per photo)
→ Ensure WebP versions are being created
→ Use a CDN for faster delivery

ISSUE: Photos don't appear in Google Images
→ Add images to your XML sitemap
→ Verify robots.txt isn't blocking image crawling
→ Give Google time to index (can take 1-2 weeks)

TROUBLESHOOTING:
----------------

Issue: Logo not displaying
→ Check file paths match your server directory structure
→ Ensure files uploaded correctly (check file permissions)

Issue: WebP not loading
→ Normal for older browsers - PNG fallback should work
→ Verify server MIME type supports image/webp

Issue: Logo too large on mobile
→ CSS in header-with-logo.html includes responsive breakpoints
→ Adjust max-width values if needed

Issue: Social media preview shows wrong image
→ Clear Facebook/LinkedIn cache using their debugger tools
→ Verify og:image URL is absolute (includes https://)
